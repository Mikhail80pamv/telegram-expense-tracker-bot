# Telegram-бот для учёта расходов

Бот помогает вести учёт личных расходов прямо в Telegram: добавлять траты по категориям и просматривать статистику за день, неделю или месяц.

## 
🚀 Установка и запуск

1. **Клонируйте репозиторий**
2. 
   ```bash
   
   git clone https://github.com/ваш_логин/имя_репозитория.git
   
   cd имя_репозитория
   
   Установите зависимости
   
   pip install -r requirements.txt
   
   Настройте токен бота

Создайте файл .env в корне проекта.

Добавьте строку: 
BOT_TOKEN=ваш_токен_от_BotFather

Запустите бота

python bot.py

📋 Команды бота

Команда	Описание

/start	Приветствие и краткая инструкция

/add	Добавить новый расход (сумма + категория)

/stats	Показать статистику расходов за неделю

/cancel	Отменить текущее действие

🛠️ Используемые технологии

Python 3.10+

Aiogram 3.x

SQLite3

python-dotenv
