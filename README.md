# Telegram Bot для учёта расходов

Бот позволяет добавлять расходы по категориям и просматривать статистику.

## Установка
1. Клонировать репозиторий.
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с токеном бота.
4. Запустить: `python bot.py`

## Команды
- `/start` – приветствие
- `/add` – добавить расход
- `/stats` – статистика
- `/cancel` – отмена
